# XRP Whale Feed

Monitors the XRP Ledger in real time and displays any transfer >= 1,000,000 XRP as it happens. Includes a live price pulled from CoinGecko.

## Stack

- Backend: Node.js + Express + WebSocket (XRPL) + SSE
- Frontend: single index.html, no framework



## Deployed on

- Backend: Railway
- Frontend: Vercel

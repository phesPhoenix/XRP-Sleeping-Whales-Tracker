about

Hey — I'm Phe, a junior CS student at Cal Poly Humboldt building toward a career in software and tech. I care about systems that actually work, backends that scale, and code that ships.

Right now I'm deep in Elementalist, an MMO project built in Unreal Engine 5. I handle the server-side infrastructure — PlayFab, Azure, the stuff that keeps the world alive. Got ping from 315ms down to 24ms. That kind of thing matters to me.

I'm also drawn to computational neuroscience and neurotech — the overlap between how brains process information and how we model that computationally. Still early, but it's the direction things are pulling.

When I'm not coding I'm usually out with a camera somewhere in the wilderness. Quiet mountains. Long exposures. No people.
